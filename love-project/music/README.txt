Add your MP3 files here, then match the filenames used in index.html:

  music/song1.mp3  -> "Panaginip" card
  music/song2.mp3  -> "Add Our Next Song" card
  music/song3.mp3  -> "Add Another Favorite" card

For legal/copyright reasons this project ships without any actual audio files.
Add your own legally-owned MP3s (e.g. exported from a CD you own, a royalty-free
track, or a song you have the rights to use) into this folder with the names
above, and the built-in player will pick them up automatically — no internet
connection required.
